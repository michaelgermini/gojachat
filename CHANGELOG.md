# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup with monorepo structure
- Real-time messaging functionality with Socket.io
- Web application with React and TypeScript
- Mobile application with React Native and Expo
- Mocked API with JSON Server
- Redux Toolkit for state management
- Comprehensive testing setup with Jest
- CI/CD pipeline with GitHub Actions
- Code quality tools (ESLint, Prettier)
- Complete documentation in English

### Changed
- Translated all French content to English
- Updated all JSON data to use English messages
- Improved TypeScript type definitions
- Enhanced error handling and user feedback
- Optimized build configuration

### Fixed
- React Router future flag warnings
- Source map generation issues
- Path alias resolution problems
- TypeScript type inference issues
- Dependency conflicts

## [1.0.0] - 2024-01-15

### Added
- Initial release of GojaChat
- Complete messaging application with web and mobile clients
- Real-time communication features
- User authentication system
- Conversation management
- Push notifications
- Responsive design
- Comprehensive documentation

### Features
- **Web Application**
  - Real-time chat interface
  - Conversation threads
  - User management
  - Responsive design
  - Push notifications

- **Mobile Application**
  - Native interface
  - Real-time synchronization
  - Offline mode support
  - File sharing capabilities

- **API**
  - RESTful endpoints
  - WebSocket support
  - User authentication
  - Message persistence

### Technical Stack
- **Frontend**: React 18, TypeScript, Redux Toolkit
- **Mobile**: React Native, Expo
- **Backend**: JSON Server, Socket.io
- **Styling**: Tailwind CSS
- **Testing**: Jest, React Testing Library
- **CI/CD**: GitHub Actions, Vercel, EAS
